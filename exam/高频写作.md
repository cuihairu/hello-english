## 二、主题分类与趋势

### 1. 环保类（出现频率：高）

**典型题目：**
- 2015：How Can We Contribute to Environmental Protection?
- 2025 翻译主题也是户外环保

**高频表达：**
- environmental protection / conservation
- reduce energy consumption
- sustainable development
- carbon footprint
- eco-friendly lifestyle

**写作框架：**
1. 引出环保的重要性
2. 提出具体做法（2-3 个）
3. 举例说明
4. 总结呼吁

### 2. 哲理/人生观类（出现频率：高）

**典型题目：**
- 2023：Is Winning What Counts Most in Our Life?
- 2019：Ways to Live a Simpler Life

**高频表达：**
- From my perspective / In my opinion
- It is widely acknowledged that...
- There is no denying that...
- What matters most is...
- lead a meaningful life

**写作框架：**
1. 引入话题（现象或观点）
2. 表明立场
3. 论证理由（2-3 个论点）
4. 总结升华

### 3. 科技/社会类（出现频率：中）

**典型题目：**
- 2021：Can Machine Translation Replace Human Translators?

**高频表达：**
- With the development of technology...
- have a profound impact on
- cannot be replaced by
- advantage and disadvantage
- speed, accuracy and performance

**写作框架：**
1. 描述现象/趋势
2. 分析利弊或表达观点
3. 举例论证
4. 总结预测

### 4. 社会公共类（出现频率：中）

**典型题目：**
- 2018：The Ideal Public Library
- 2017：What Makes Happy Couples Happy?

## 三、高频写作结构

### 结构一：观点论述型（最常见）

```
第一段（引言）：
  引入话题 → 表明观点

第二段（正文 1）：
  论点一 + 解释 + 举例

第三段（正文 2）：
  论点二 + 解释 + 举例

第四段（结论）：
  总结重申观点 → 展望/呼吁
```

### 结构二：方法建议型

```
第一段（引言）：
  描述问题/现状 → 引出主题

第二段（方法一）：
  具体建议 + 理由

第三段（方法二）：
  具体建议 + 理由

第四段（结论）：
  总结方法 → 表达期望
```

## 五、过渡词/连接词

### 添加信息
- in addition, furthermore, moreover, what's more, besides, also

### 表示转折
- however, nevertheless, on the contrary, whereas, although, despite

### 表示因果
- therefore, consequently, as a result, thus, hence, due to, owing to

### 表示举例
- for example, for instance, such as, take... as an example, namely

### 表示顺序
- first(ly), second(ly), third(ly), finally, last but not least, to begin with

### 表示总结
- in conclusion, to sum up, all in all, in brief, on the whole

## 六、写作高频同义词

这些词不要求全部生僻，重点是把最常用、最稳妥、最容易替换的表达练熟。

### 1. 表示“重要”

| 常用表达 | 可替换表达 |
|------|------|
| important | crucial, significant, vital, essential |
| very important | of great importance, extremely significant |
| play an important role in | play a crucial role in, be essential to |

### 2. 表示“认为”

| 常用表达 | 可替换表达 |
|------|------|
| think | believe, hold that, maintain that |
| I think | I believe, in my opinion, from my perspective |
| many people think | it is widely believed that, many people hold the view that |

### 3. 表示“导致”

| 常用表达 | 可替换表达 |
|------|------|
| cause | lead to, result in, bring about, give rise to |
| make | enable, allow, help, contribute to |
| influence | affect, have an impact on |

### 4. 表示“好/坏”

| 常用表达 | 可替换表达 |
|------|------|
| good | beneficial, positive, favorable |
| bad | harmful, negative, unfavorable |
| helpful | useful, beneficial, effective |
| harmful | damaging, destructive, unfavorable |

### 5. 表示“越来越……”

| 常用表达 | 可替换表达 |
|------|------|
| more and more important | increasingly important |
| more and more people | a growing number of people |
| become more serious | become increasingly serious |
| more and more common | more prevalent, increasingly common |

### 6. 表示“解决问题”

| 常用表达 | 可替换表达 |
|------|------|
| solve the problem | address the problem, tackle the problem |
| deal with | cope with, handle |
| improve | enhance, promote |
| reduce | decrease, lower, cut down on |

### 7. 表示“得到好处/产生结果”

| 常用表达 | 可替换表达 |
|------|------|
| be good for | be beneficial to, contribute to |
| be bad for | do harm to, be harmful to |
| get benefits | gain benefits, reap the benefits |
| get success | achieve success |

## 七、写作高频同义短句

这些短句比单个词更实用，适合直接放进作文。

### 1. 开头引题

| 基础写法 | 升级写法 |
|------|------|
| Now, ... is important. | Nowadays, ... has become increasingly important. |
| People care about ... | ... has aroused widespread concern. |
| Many people think that ... | It is widely believed that ... |
| Different people have different ideas. | Opinions vary from person to person. |

### 2. 表达个人观点

| 基础写法 | 升级写法 |
|------|------|
| I think ... | As far as I am concerned, ... |
| I think ... is important. | From my perspective, ... is of great importance. |
| I agree that ... | I am in favor of the view that ... |
| I don't think ... | I do not believe that ... |

### 3. 展开理由

| 基础写法 | 升级写法 |
|------|------|
| First, ... | First and foremost, ... |
| Second, ... | In addition, ... / Furthermore, ... |
| Also, ... | What's more, ... / Moreover, ... |
| Because ... | This is because ... / The reason is that ... |

### 4. 举例说明

| 基础写法 | 升级写法 |
|------|------|
| For example, ... | For instance, ... |
| Let's take ... as an example. | Take ... as an example. |
| This shows that ... | This clearly shows that ... |

### 5. 表示结果与影响

| 基础写法 | 升级写法 |
|------|------|
| So, ... | As a result, ... / Therefore, ... |
| It is good for ... | It is beneficial to ... |
| It is bad for ... | It is harmful to ... |
| It changes our life. | It has a profound impact on our lives. |

### 6. 结尾总结

| 基础写法 | 升级写法 |
|------|------|
| In conclusion, ... | To sum up, ... / All in all, ... |
| We should ... | It is necessary for us to ... |
| Only like this can we ... | Only in this way can we ... |
| I believe ... | I am firmly convinced that ... |

### 7. 可直接套用的万能短句

- There is no denying that ...
- It is universally acknowledged that ...
- One of the main reasons is that ...
- This not only ... but also ...
- It is high time that we took effective measures to ...
- Only in this way can we make a difference.
- Every coin has two sides.
- We should make full use of its advantages and avoid its disadvantages.

## 八、作文万能替换表

这部分适合考前集中背诵。原则不是一味追求生词，而是把口语化、基础化表达稳定替换成更书面、更安全的写法。

### 1. 高频基础词三级替换

| 基础表达 | 稳妥替换 | 更书面替换 |
|------|------|------|
| important | important / essential | crucial / of great importance |
| good | good / helpful | beneficial / favorable |
| bad | bad / harmful | negative / unfavorable |
| big | large | considerable / significant |
| more and more | increasingly | progressively |
| many | many | a large number of / a growing number of |
| think | think / believe | hold the view that / maintain that |
| show | show | demonstrate / indicate |
| cause | cause | lead to / result in / give rise to |
| solve | solve | address / tackle |
| improve | improve | enhance / promote |
| use | use | make use of / take advantage of |
| get | get | gain / obtain / acquire |
| help | help | assist / contribute to |
| important role | important role | crucial role / essential part |

### 2. 常见口语句替换

| 不够稳妥的写法 | 更推荐的写法 |
|------|------|
| I think | In my opinion / As far as I am concerned |
| I think it is very important | I believe it is of great importance |
| More and more people think | A growing number of people believe |
| It can make people better | It can help people improve themselves |
| It is good for society | It is beneficial to society |
| It is bad for students | It is harmful to students |
| We should use it well | We should make proper use of it |
| We should solve this problem | We should take effective measures to address this problem |
| This thing changes our life | This phenomenon has a profound impact on our lives |
| We can only do it this way | Only in this way can we achieve the goal |

### 3. 段落功能替换

| 写作功能 | 基础写法 | 推荐写法 |
|------|------|------|
| 引出话题 | Now ... is important | Nowadays, ... has become a topic of growing concern |
| 表明观点 | I think ... | From my perspective, ... |
| 提出第一点 | First | First and foremost |
| 补充第二点 | Second / Also | In addition / Furthermore |
| 举例 | For example | For instance / Take ... as an example |
| 总结 | In conclusion | To sum up / All in all |

## 九、按主题分类的高频句库

以下句子尽量控制在考试里能直接改写、直接套用的范围。

### 1. 环保主题

- Environmental protection is not only the responsibility of governments but also the duty of every individual.
- Small actions, such as saving water and reducing waste, can make a big difference.
- Developing an eco-friendly lifestyle is beneficial to both human beings and the planet.
- It is necessary for us to raise public awareness of environmental protection.
- Only when people take practical action can the environment be effectively improved.

### 2. 教育与学习主题

- Lifelong learning has become increasingly important in the modern world.
- Continuous learning enables people to keep pace with social and technological changes.
- Education not only broadens our minds but also improves our ability to solve problems.
- A good learning habit can lay a solid foundation for future success.
- It is widely believed that learning should be a lifelong process rather than a short-term task.

### 3. 科技与人工智能主题

- With the rapid development of technology, artificial intelligence is having a profound impact on our daily life.
- Technology brings convenience and efficiency, but it also creates new challenges.
- No matter how advanced machines become, some human qualities cannot be easily replaced.
- We should make full use of technology while avoiding its potential risks.
- It is essential to strike a balance between technological progress and ethical responsibility.

### 4. 人生观与价值观主题

- Success should not be measured only by wealth or victory.
- What matters most is whether a person can live a meaningful and fulfilling life.
- A positive attitude often plays a crucial role in personal growth.
- Instead of comparing ourselves with others, we should focus on self-improvement.
- True happiness comes from inner satisfaction rather than external competition.

### 5. 健康生活主题

- A healthy lifestyle is of great importance to both physical and mental well-being.
- Regular exercise and a balanced diet are essential to good health.
- It is necessary for people to maintain a balance between work and life.
- Too much pressure may do harm to both our body and mind.
- Only by forming healthy habits can we enjoy a better quality of life.

### 6. 社会与公共生活主题

- Public services play an essential role in improving people's quality of life.
- A well-functioning public space can meet both practical and cultural needs.
- Mutual understanding and cooperation are important in building a harmonious society.
- Social progress depends not only on economic growth but also on public responsibility.
- It is important to create an environment in which everyone feels respected and supported.

### 7. 可跨主题通用的结尾句

- In conclusion, the issue deserves our serious attention.
- Taking all these factors into account, we should adopt a rational attitude toward it.
- Only in this way can we achieve sustainable and long-term development.
- I am firmly convinced that a better future can be created through joint efforts.
- If effective measures are taken, the situation will be greatly improved.

## 十、主题范文参考

以下范文尽量控制在 150-180 词之间，句式以稳妥、可套用、易改写为主。

### 1. 教育主题

**题目：** The Importance of Lifelong Learning

```text
There is no denying that lifelong learning has become increasingly important in modern society. As far as I am concerned, continuous learning is essential to both personal growth and social development.

First of all, lifelong learning helps people keep pace with the rapid development of society. New knowledge and new technology appear every day, and only by learning continuously can we avoid falling behind. In addition, lifelong learning broadens our minds and improves our ability to solve practical problems. It also creates more opportunities for us in work and in daily life.

Last but not least, learning is not only a way to gain knowledge but also a way to enrich our lives. A person who keeps learning is more likely to remain confident, open-minded and adaptable.

In conclusion, lifelong learning is of great importance to everyone. We should develop the habit of learning throughout life and make constant progress.
```

### 2. 科技主题

**题目：** Can Artificial Intelligence Replace Human Beings?

```text
With the rapid development of science and technology, artificial intelligence has become a hot topic. Some people believe that AI will replace human beings in many fields. However, as far as I am concerned, AI can assist us, but it cannot completely replace human beings.

On the one hand, AI has many obvious advantages. It can process information quickly, improve work efficiency and reduce repetitive labor. For example, in some routine jobs, machines can work faster and more accurately than humans.

On the other hand, human beings still have unique qualities such as creativity, emotion and moral judgment. These qualities play a crucial role in education, art and many social activities. No matter how advanced machines become, they cannot fully understand human feelings or values.

In conclusion, AI is a powerful tool rather than a complete substitute for human beings. We should make proper use of it while recognizing the irreplaceable value of humans.
```

### 3. 环保主题

**题目：** How Can We Contribute to Environmental Protection?

```text
Environmental protection has aroused widespread concern in recent years. Since the environment is closely related to our daily life, it is necessary for everyone to take action.

To begin with, we should develop environmentally friendly habits. For example, we can save water and electricity, use fewer plastic products and sort household waste properly. In addition, choosing public transportation or riding bicycles is also an effective way to reduce pollution.

Moreover, we should raise our awareness of environmental protection and encourage others to join us. Small actions may seem unimportant, but together they can make a big difference. If more people realize the importance of protecting nature, our environment will gradually improve.

In conclusion, environmental protection depends on the efforts of us all. Only by taking practical steps in our daily life can we build a cleaner and better world.
```

### 4. 人生观主题

**题目：** Is Winning What Counts Most in Our Life?

```text
Different people have different opinions about success and winning. Some people believe that winning is the most important thing in life. However, I do not think so.

Admittedly, winning can bring people confidence and a sense of achievement. It may also encourage people to work harder and improve themselves. However, if we care too much about winning, we may become anxious, selfish or even lose the true meaning of life.

In my opinion, what matters most is not whether we always win, but whether we grow through our experiences. A meaningful life should include honesty, responsibility, friendship and inner satisfaction. Even if we fail sometimes, we can still learn valuable lessons and become better people.

To sum up, winning is important, but it is not everything. What really counts is how we face life, treat others and continue to improve ourselves.
```

### 5. 健康生活主题

**题目：** The Importance of a Healthy Lifestyle

```text
A healthy lifestyle is of great importance to everyone. In modern life, many people are under great pressure and often ignore their health. Therefore, it is necessary for us to develop good living habits.

First, regular exercise helps us build a strong body and stay energetic. Second, a balanced diet provides the nutrition our body needs and helps prevent disease. Besides, getting enough sleep is also essential, because it allows both the body and the mind to recover from fatigue.

More importantly, a healthy lifestyle benefits not only our physical condition but also our mental well-being. People who live healthily are often more positive and efficient in their work and study.

In conclusion, health is the foundation of a happy life. Only by forming healthy habits can we enjoy a higher quality of life and a better future.
```

### 6. 社会公共主题

**题目：** The Ideal Public Library

```text
When it comes to public services, the library remains one of the most important places in a city. In my opinion, an ideal public library should be convenient, comfortable and helpful to people of all ages.

First of all, it should provide a wide variety of books, including literature, science, history and children's reading materials. In this way, different readers can satisfy their own needs. In addition, a good library should offer a quiet and comfortable environment, so that people can read, study and think peacefully.

Moreover, modern libraries should not be limited to traditional reading functions. They can also provide digital resources, study spaces and public lectures, which make learning more accessible and efficient.

In conclusion, an ideal public library is not only a place for reading but also an important center for education and culture. It should serve the public in a comprehensive way.
```

## 十一、冲刺背诵版

这一部分只保留最值得直接背诵的内容，适合考前 1-3 天快速过一遍。

### 1. 考场速用 30 句

#### 开头句

1. There is no denying that this issue has become increasingly important.
2. Nowadays, this topic has aroused widespread concern.
3. With the rapid development of society, this issue has become a hot topic.
4. Different people hold different views on this issue.
5. As far as I am concerned, this issue is of great importance.

#### 观点句

6. From my perspective, the advantages outweigh the disadvantages.
7. I am in favor of the view that we should take this issue seriously.
8. I do not believe that this problem can be solved in a simple way.
9. What matters most is how we deal with this issue.
10. It is widely believed that this issue deserves our attention.

#### 正文展开句

11. First and foremost, this can bring many benefits to individuals and society.
12. In addition, it helps improve efficiency and quality.
13. What's more, it enables people to make better choices.
14. One of the main reasons is that it can meet practical needs.
15. Another important reason is that it contributes to long-term development.
16. For instance, many people have already benefited from such changes.
17. Take our daily life as an example, we can easily see its influence.
18. This clearly shows that the issue should not be ignored.
19. Small actions may seem unimportant, but together they can make a big difference.
20. We should not only see the benefits but also recognize the possible problems.

#### 结果与影响句

21. As a result, people can enjoy a better quality of life.
22. Therefore, it has a profound impact on both individuals and society.
23. If left unsolved, the problem may become increasingly serious.
24. In this way, we can create a more positive and efficient environment.
25. Only by taking effective measures can we improve the situation.

#### 结尾句

26. In conclusion, this issue deserves our serious attention.
27. To sum up, we should adopt a rational attitude toward it.
28. It is necessary for us to take practical action.
29. Only in this way can we achieve sustainable development.
30. I am firmly convinced that a better future can be created through joint efforts.

### 2. 五段式万能开头

#### 模板一：观点论述型

```text
There is no denying that [话题] has become a topic of growing concern in modern society. Different people may have different opinions on it. As far as I am concerned, [你的观点].
```

#### 模板二：问题解决型

```text
With the rapid development of society, [问题] has become increasingly serious. It is necessary for us to take effective measures to deal with it.
```

#### 模板三：利弊分析型

```text
Nowadays, [话题] is playing an increasingly important role in our daily life. Like many other things, it has both advantages and disadvantages.
```

### 3. 五段式万能结尾

#### 模板一：总结观点

```text
In conclusion, [重申观点]. Therefore, we should attach greater importance to this issue and take practical action.
```

#### 模板二：呼吁行动

```text
To sum up, the problem cannot be solved in one day, but if we work together and take effective measures, the situation will be greatly improved.
```

#### 模板三：升华总结

```text
All in all, [话题] is of great importance to both individuals and society. Only in this way can we create a better future.
```

## 十三、备考建议

1. **背诵模板句型**：熟练掌握引言、正文、结论段的高频句型
2. **背同义词和同义短句**：把基础词替换成更稳妥的书面表达
3. **背万能替换表**：优先替换 `important / good / bad / think / cause / solve`
4. **按主题背句子**：环保、教育、科技、人生观至少各背 5 句
5. **练习三大类主题**：环保、哲理、科技各准备一篇范文
