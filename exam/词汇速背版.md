# 词汇速背版

> 适用场景：考前 10 分钟快速浏览，优先抓住高频同义替换和固定短语。

## 一、词汇题核心规律

- 词汇题本质上是同义替换题
- 动词、形容词、短语最常考
- 不只考单词意思，更考固定搭配和语境
- 真题里“短语替换”占比稳定，不可忽视

## 四、高频语义组

这一部分最值得背，因为题目经常换词不换义。

### 1. 停止 / 取消

- cease -> stop
- suspend -> halt
- call off -> cancel
- wear off -> disappear

### 2. 充足 / 资格

- sufficient -> enough
- ample -> enough
- eligible -> qualified

### 3. 解释 / 展示

- demonstrate -> show
- elaborate -> explain
- put across -> explain

### 4. 因果 / 归因

- give rise to -> cause
- attribute to -> cause
- consist in -> lie in

### 5. 态度 / 倾向

- deliberate -> intentional
- on purpose -> deliberately
- inclined -> leaning
- reluctantly -> unwillingly

## 六、最容易混的点

### 1. consist in / consist of

- consist in: 在于
- consist of: 由……组成

### 2. give out

- 真题里常考“泄露”
- 不要只背“分发”

### 3. eligible

- 常和 `for` 连用
- 表“有资格的”

### 4. deliberate / deliberately

- deliberate: 故意的
- deliberately / on purpose: 故意地

### 5. critical

- 真题里常不是“批评的”
- 更常考“关键的” = crucial
