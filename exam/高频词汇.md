## 三、备考建议

### 1. 动词是核心（约占 40%）

重点关注**同义动词替换**，如：
- 表"停止"：cease/stop/halt/suspend
- 表"超越"：surpass/exceed/go beyond
- 表"解释"：elaborate/explain/clarify
- 表"评估"：assess/evaluate/judge

### 2. 形容词/副词占比大（约占 35%）

高频考查的形容词对：
- sufficient / enough
- deliberate / intentional
- eligible / qualified
- substantial / significant
- crucial / essential

### 3. 短语辨析（约占 25%）

常考的动词短语：
- give rise to / lead to / result in
- carry out / implement / perform
- wear off / disappear / fade away
- find fault with / criticize / blame

### 4. 复现词汇（多次考查）

以下词汇在不同年份重复出现：
- **eligible** → qualified（2016、2025）
- **collaborate** → cooperate（2023、2025）
- **sufficient** → enough（相关概念多年出现）

### 5. 备考策略

1. **熟记常见同义词对**：本文件中的表格覆盖了历年真题中所有考查过的替换关系
2. **关注语境**：不是简单的词义记忆，而是在句子中理解词义
3. **注意陷阱选项**：干扰项常与画线词有形似或部分语义重叠，但不是最佳替换
4. **动词短语**：每年至少 2-3 题考查短语，建议系统整理常见动词短语
